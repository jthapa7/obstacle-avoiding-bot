# obstacle-avoiding-bot

## Required Parts
1. Ultrasonic Sensor
2. Arduino UNO
3. Adafruit Motor Sheild
4. DC Motors
5. Servo
6. Chasis

## Library Used
Adafruit_MotorShield library was used to drive the motors and the servos
